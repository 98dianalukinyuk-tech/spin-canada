# Spin Casino physical pokerroom guide — Canada

Locale: `en-CA`  
Currency for verified event data: CAD  
Time zone: use the verified venue's local Canadian time zone.

## Run locally

From the project folder:

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080/`.

## Verified data

Replace the technical markers in `script.js` before publication:

- `{{VERIFIED_BASE_URL}}`
- `{{VERIFIED_ADDRESS}}`
- `{{VERIFIED_PHONE}}`
- `{{VERIFIED_EMAIL}}`
- `{{VERIFIED_LEGAL_AGE}}`

Populate only confirmed venue events, broadcasts and results. Keep `noindex,nofollow` and `robots.txt` blocking enabled until legal authority, venue details, privacy contacts and publication rights are confirmed.

## Positioning

This project is an independent information guide for physical pokerroom visits. It does not provide online poker, gambling accounts, deposits, withdrawals, wallets, balances, bonuses or remote tournament participation.
